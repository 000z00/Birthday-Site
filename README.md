Place your files in these exact locations:

## Images
```
Bsite/images/
├── memory1.jpg   ← Timeline Memory 1 photo
├── memory2.jpg   ← Timeline Memory 2 photo
├── memory3.jpg   ← Timeline Memory 3 photo
├── memory4.jpg   ← Timeline Memory 4 photo
├── memory5.jpg   ← Timeline Memory 5 photo
├── letter1.jpg   ← Letter 1 photo (Remember This Day)
├── letter2.jpg   ← Letter 2 photo (The Dumbest Moment)
├── letter3.jpg   ← Letter 3 photo (Favorite Memory)
├── letter4.jpg   ← Letter 4 photo (Time You Helped Us)
├── letter5.jpg   ← Letter 5 photo (Why We're Still Friends)
└── letter6.jpg   ← Letter 6 photo (Birthday letter)
```

## Video
```
Bsite/videos/
└── birthday.mp4  ← Birthday video (uncomment in index.html)
```

## Music (optional)
```
Bsite/music/
└── background.mp3  ← Background music (uncomment in index.html)
```

## How to Enable Video
Uncomment lines 195–199 in index.html and remove the placeholder div above it.

## How to Enable Music
Uncomment lines 202–205 in index.html and add autoplay logic in script.js.
